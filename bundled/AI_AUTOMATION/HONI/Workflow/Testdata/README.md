# Testdata

Put **workflow test data importer** files and related assets in this folder (for example, the file name you reference in each test case’s **Test data** section).

- Keep **client-specific or secret** data out of source control if your team policy requires it.
- Test cases under `../Testcase/` can refer to importers by name, e.g. `Workflow test data importer named <your-file> is being used.`
